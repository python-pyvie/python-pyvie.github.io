Pyvie is a Python module designed to close the gap in movie making and animation in Python.

It was written by David J. Butts after wanting to automate his workflow for making videos in Python. Pyvie streamlines the process of making a series of image files and stitching them together using FFMPEG, eliminating the need to image files specific names, and leave Python scripts to create a video. It can open the videos it makes within a Jupyter notebook or a users default media player, allowing to quickly view movie file without having to search for them.
