About

 Mon 01 January 2018 by

Pyvie is a Python module designed to close the gap in movie making and animation in Python. It streamlines the movie making process in Python with the use of FFMPEG in a little as four lines.

It was written with the goal of automating making videos in Python. Pyvie streamlines the process of making a series of image files and stitching them together using FFMPEG, eliminating much of the hassle that comes with that workflow. The user no longer needs to think about naming image files specific names, leaving Python scripts to create a video, or remembering long FFMPEG arguments. Pyvie can open the videos it makes within a Jupyter notebook or a users default media player, allowing to quickly view movie file without having to search for them. It also allows for users to reopen movies that have previously created.
